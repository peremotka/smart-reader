# Python Сервис (Анализатор Текста)

Микросервис на FastAPI для анализа текста.

## Локальный запуск (через терминал в Ubuntu)

Если нужно запустить сервис вручную из консоли, выполните следующие шаги:

1. **Перейдите в папку с Python-кодом** (если вы еще не в ней):
   ```bash
   cd ~/IdeaProjects/smart-reader/python

2. **Активируйте виртуальное окружение (venv)**:
   ```bash
   source venv/bin/activate

3. После этого в начале строки терминала должны появиться скобочки **(venv).**

4. Запустите сервер разработки Uvicorn:
   ```bash
   uvicorn main:app --reload

5. Сервер поднимется на адресе: http://127.0.0.1:8000
6. Для остановки сервера нажмите в терминале: Ctrl + C
